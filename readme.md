# Football Ticket Booking System


This project implements a simple relational database for a **Football Ticket Booking System** using SQL.

The database stores:

* Users (Football Fans and Ticket Managers)
* Football Matches
* Ticket Bookings

## Files

* `Football_Ticket_Booking_System.sql` – Creates the database tables and inserts sample data and Contains the required SQL queries for the assignment.


## Database Tables

* **Users** – Stores registered users.
* **Matches** – Stores football match information.
* **Bookings** – Stores ticket booking records and links users with matches.

## Features

* Primary Keys
* Foreign Keys
* UNIQUE Constraints
* CHECK Constraints
* Sample Data
* SQL Queries using:

  * SELECT
  * WHERE
  * LIKE / ILIKE
  * COALESCE
  * JOIN
  * LEFT JOIN
  * Aggregate Functions
  * Subqueries
  * ORDER BY, LIMIT, OFFSET
